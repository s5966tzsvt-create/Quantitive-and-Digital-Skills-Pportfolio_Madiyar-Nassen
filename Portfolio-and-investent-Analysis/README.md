# Project 1 – Portfolio & Investment Analysis
## Quantitative and Digital Skills Portfolio

### Overview
This project analyses the daily risk-return profile of two large-cap technology stocks —
Alphabet Inc. (GOOGL) and Meta Platforms Inc. (META) — using daily adjusted closing
price data sourced from Yahoo Finance (October 2024 – January 2025, 58 trading days).

---

### Files
- Portfolio_GOOGL_META_Full.xlsx — Main analytical spreadsheet
- Synopsis.pdf — One-page professional synopsis
- Skills_Paragraph.txt — Skills reflection paragraph
- README.md — This file

---

### Spreadsheet Structure

| Sheet       | Contents                                              |
|-------------|-------------------------------------------------------|
| co1         | GOOGL daily OHLCV raw data (Yahoo Finance)            |
| co2         | META daily OHLCV raw data (Yahoo Finance)             |
| returns     | Arithmetic and geometric daily returns                |
| descr       | Descriptive statistics (mean, std dev, IQR, MAD etc.) |
| var dev     | Variance and deviation comparison                     |
| cvar        | Coefficient of Variation (CV = Std Dev ÷ Mean)        |
| z outliers  | Z-scores and outlier identification ( |Z| > 3 )       |
| cov         | Covariance and correlation (manual + Excel functions) |
| sharpe      | Sharpe Ratio — daily and annualised                   |
| synopsis    | Summary statistics table and written synopsis​​​​​​​​​​​​​​​​
